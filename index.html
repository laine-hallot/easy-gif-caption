Add text to gif

Info:

Easy gif Caption allows you to add a caption on pretty much any gif.

The layout of the resulting gif is modeled after Assyst's -caption function (idk if that's where the particular black on white top caption format came from but that's what I've always used to make them ¯\_(ツ)_/¯)

Everything is done on your device using ffmpegwasm which keeps this site's hosting costs low ($0), meaning there's no ads.
